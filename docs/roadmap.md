## Фаза 2: Fine-tuning (сентябрь — октябрь 2026)

### Техническая часть
1. **LoRA fine-tuning whisper-small** на собранном датасете
   - База: openai/whisper-small (244M params)
   - LoRA: r=32, α=64, target_modules=q_proj,k_proj,v_proj,out_proj
   - Trainable: ~7M params (2.84%)
   - Оборудование: MacBook M1 Max 64GB (MPS backend)
   - Либо аренда A100 (около 30-50 ₽/час на TimeGPU/Cloud.ru)
2. **Оценка качества (WER/CER)**
   - Baseline zero-shot → LoRA fine-tuned
   - Цель: WER < 30% на тестовом сете
3. **Итеративное улучшение**
   - Добавление новых данных по худшим сэмплам
   - Data augmentation (speed pert., noise, SpecAugment)
4. **Повтор для второй модели** (small → medium, если данные позволяют)
5. **Публикация модели** на Hugging Face Hub

### Бюджет фазы 2: ~50 000 ₽ (на оба проекта)
- 30 000 ₽ — аренда GPU (около 100 часов A100)
- 10 000 ₽ — валидация, тестирование
- 10 000 ₽ — LabelStudio развёртывание, хранение данных

### Результат
- [ ] Whisper-small-os: fine-tuned модель для осетинского (WER < 30%)
- [ ] Whisper-small-ce: fine-tuned модель для чеченского (WER < 30%)
- [ ] Hugging Face модель опубликована
- [ ] Baseline vs fine-tuned: отчёт с метриками

## Риски и митигация

| Риск | Вероятность | Влияние | Митигация |
|---|---|---|---|
| Недостаточно данных для осетинского | Средняя | Высокое | Common Voice + MMS-lab + YouTube |
| Нет данных для чеченского | Высокая | Высокое | CMU Wilderness + MMS TTS синтез + IT-куб |
| Yandex/Sber выпустят ASR раньше | Средняя | Среднее | Open-source + offline — наше УТП |
| Special tokens в whisper.cpp | Высокая | Среднее | Форк whisper.cpp |
| Оборудование Точки Роста слабое | Низкая | Низкое | Whisper small работает на любом CPU |
| Школьники не хотят участвовать | Низкая | Среднее | Мотивация: сертификаты, гранты, поступление |

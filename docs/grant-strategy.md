# Грантовая стратегия
## Финансирование проекта Whisper STT для осетинского и чеченского

**Дата:** 23 июня 2026
**Стартовый бюджет (личный):** 50 000 ₽ на каждый проект
**Целевое привлечение:** до 20 млн ₽ суммарно

---

## Стратегический подход

Проект финансируется по принципу **«пирамиды источников»**:

```
        ┌──────────────────┐
        │ Крупные гранты   │  ← ПФКИ, ФПГ, РНФ (до 20 млн ₽)
        │  3-20 млн ₽      │
        ├──────────────────┤
        │ Средние гранты   │  ← Росмолодёжь, ФСИ, региональные (до 5 млн ₽)
        │  0.5-5 млн ₽     │
        ├──────────────────┤
        │ Быстрые деньги   │  ← УМНИК, Премия Главы, «Ключевое слово» (до 500 тыс)
        │  до 500 тыс ₽    │
        ├──────────────────┤
        │ Свои средства    │  ← 50 000 ₽ на проект (Фаза 1)
        │  50 000 ₽        │
        └──────────────────┘
```

Каждый уровень:
- **Свои средства** — запуск (Фаза 0-1)
- **Быстрые деньги** — подпитка на Фазе 2-3 (без сложной отчётности)
- **Средние гранты** — Фаза 3-4 (основной бюджет разработки)
- **Крупные гранты** — Фаза 5 (масштабирование на 3+ языка)

## 8. Шаблоны обоснований для заявок

### Для ПФКИ / ФПГ (социальный эффект):
> «Осетинский язык находится в статусе "уязвимого" по классификации ЮНЕСКО. Число владеющих сократилось с 494 тыс. (2002) до 345 тыс. (2020). Ключевая причина — отсутствие языковой среды в цифровом пространстве. Проект создаёт первый открытый движок распознавания устной осетинской речи, что позволяет:
> - Разрабатывать голосовые интерфейсы на осетинском
> - Создавать субтитры к осетинскому контенту
> - Использовать технологию в образовательных целях в школах
> Проект реализуется на базе «Точки Роста» СОШ п. Верхний Фиагдон и ДТ «Кванториум-15» при поддержке Фонда цифрового развития осетинского языка.»

### Для ФСИ / УМНИК (технологическая новизна):
> «Разрабатывается система автоматического распознавания осетинской/чеченской речи на основе fine-tuning архитектуры Whisper (OpenAI). Уникальность:
> - Первая open-source ASR-модель для данных языков
> - Работает offline (без интернета)
> - Поддерживает код-свитчинг (русский + целевой язык)
> - Модель конвертируется в ggml-формат для whisper.cpp
> Техническая новизна: добавление новых языков в архитектуру Whisper, не поддерживающих стандартными 99 языками OpenAI.»

### Для РНФ (научная значимость):
> «Проект направлен на решение задачи включения нового языка в архитектуру Whisper без переобучения всей модели. Разрабатывается методология LoRA-адаптации для индо-иранских (осетинский) и нахско-дагестанских (чеченский) языков. В результате:
> - Методика расширения Whisper на новые языки
> - Первый датасет осетинской/чеченской речи в открытом доступе
> - Бенчмарк качества распознавания для малоресурсных языков РФ»
